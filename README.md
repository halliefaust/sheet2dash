# Dashify

## How to Run

### 1. Start the Backend

Run the following commands in your terminal:

```bash
export OPENAI_API_KEY=[YourOpenAPIKey]
export GOOGLE_API_KEY=[YourGoogleAPIKey]
```

```bash
python server.py
```

### 2. Start the Frontend

Open a new terminal window and enter:

```bash
npm run dev
```

### 3. Open the Application

Go to:

```
http://localhost:3000/
```
